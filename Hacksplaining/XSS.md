# Cross-Site Scripting

## What is Cross-Site Scripting
Cross-Site Scripting happens when an attacker injects malicious scripts (like JavaScript) into a trusted website, then execute in victims' browsers.
3 Types of XSS:
1.  Reflected XSS
2.  Stored XSS
3.  DOM-based XSS

## Impact of Cross-Site Scripting
-  Session Hijacking
-  Dos Attack
