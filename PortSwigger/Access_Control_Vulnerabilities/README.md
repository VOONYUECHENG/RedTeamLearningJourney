# Access Control Vulnerabilities

This folder documents my hands-on learning of access control vulnerabilities through PortSwigger Web Security Academy labs.

## Focus areas:
- Vertical privilege escalation (user to admin)
- Horizontal access control (accessing other users' data)
- Insecure Direct Object Reference (IDOR)
- Access control bypass via HTTP methods
- Parameter-based and role-based access control flaws
- Multi-step workflow bypass

## Files:
- `notes.md` – key concepts and observations
- `attack_techniques.md` – practical exploitation techniques
