# Authentication Vulnerabilities

This folder documents my hands-on learning of authentication vulnerabilities through PortSwigger Web Security Academy labs.

## Focus areas:
-  User enumeration through error messages or application behaviour.
-  Brute-force attempts.
-  Multi-factor bypass or brute-force.
-  "Stay logged in" cookie exploits.
-  Reset password function exploits.
-  Password change function exploits.

## Files:
-  `notes.md` for lab observations and lessons
-  `attack_techniques.md` for authentication vulnerabilities exploit techniques
